羽毛球

羽球教学计划

热身动作

羽毛球是对抗型的高强度运动，开打之前热身和打完之后拉伸是非常重要的。肩部、腰部、膝关节、小腿、脚踝和手腕等都需要进行热身和拉伸。羽毛球热身动作很多，常见的有：

  • 高抬腿、高抬腿内收、高抬腿外展、侧压腿、深蹲
  • 开合跳、交叉跳、十字跳、转髋跳、V字跳
  • 前后弓箭步、左右弓箭步、斜侧弓箭步
  • 深蹲转髋（拿督弹簧腿热身）

PS：这些热身动作有很多在打羽毛球的过程中都是有直接体现的。

CLASS 1

CONTENT-1：握拍

认识羽毛球球拍规格3U、4U、5U、6U；手柄规格G5、G6，了解手柄棱角：宽棱、侧棱、斜棱。学习正手握拍和反手握拍。

正手握拍

两种握拍方法：

  • 9字握拍
  • 拳式握拍

反手握拍

学会握拍之后培养球感动作：正、反手颠球

CONTENT-2：架拍框架

学习羽毛球架拍姿势、挥拍姿势和挥拍发力。 动作分解为：三步走 or 四步走；主要内容有：架拍、蹬地转髋、顶肘倒拍、小臂内旋发力、收拍。
在连贯核心动作之外，可以学习一些辅助性的单个动作作为支撑，如：

  • 立腕挥拍、直升机挥拍
  • 小臂内旋发力

架拍框架是一个肌肉记忆养成的过程，身体需要长时间且重复的去适应整个动作和发力，尤其是一些习惯了使用苍蝇拍打球的小伙伴，要想彻底的纠正架拍框架，时间周期会更长，从学习正确架拍框架，加上居家练习，实战中有意识的去使用正确姿势，一开始为了练姿势，回球质量不高甚至接不到球都是正常的，这是一个量变到质变且不断进行修正的过程。

CONTENT-3：发球

基础

了解羽毛球球场的区域划分：

  • 客观上【有明确边界线】：边线、底线；
  • 主观上【无明确边界线】：前场、中场、后场、中前场、中后场；

了解羽毛球发球规则和点位分布：1号位、2号位、3号位、4号位、5号位、6号位。
学习羽毛球发球，分为两种：

  • 【后场】高远球 正手发高远球，高远球目标落点在底线的长方形框中。
  • 【前场】网前球 反手发网前球，网前球常见的固定点位练习：1号位、5号位。

进阶

  • 学习剩余点位发球，2号位、3号位、4号位和6号位属于战术位，发这些点位的出发点是想在发球时从战术上占据主动；
  • 学习观察对手站位和握拍，选择合适的发球点位，争取在发球时就占据当前回合的主动性；

发球需要一段时间的密集重复练习来提升稳定性；

CLASS 2

CONTENT-1：高远球

高远球，是指从己方后场到对手后场都在高空飞行的球。一般分为正手高远和反手高远。

了解单打和双打的接发站位，学习中后场固定点位接发后场高远球，对拉后场高远球，后场高远球是羽毛球的实战基础，很重要，需要大量的重复练习。
内容：

  • 固定点位回高远球
  • 对拉后场
  • 后场软压

CONTENT-2：基础步法

学习羽毛球常用步伐：

  • 并歩
  • 垫步
  • 交叉步
  • 跨步
  • 启动步
  • 确认步
  • 马来步

步伐训练是最累的，也是最基础的，在羽毛球场上，如果脚步跑不到位，那么就会一直处于被动防守状态，这个也是业余球友中等级的瓶颈。
球场6点步需要长期的练习，同时也是热身的一个好平替。

CONTENT-3：移动高远球

结合基础后场步伐，练习后场步伐+高远回球。

  • 并歩退后场回高远
  • 交叉步退后场回高远
  • 马来步退后场回高远

CLASS 3

CONTENT-1：挑球

结合网前左右两点步伐，承接学习正/反手挑球。

  • 正手挑球
  • 反手跳球

挑球：后场直线高远、后场斜线线高远。

CONTENT-2：吊球

吊球，是从后场落到对手前场的球。吊球技术可分为正手、反手和头顶，根据球的飞行曲线和击球动作可分为劈吊、拦截吊和轻吊。劈挂前的动作类似于打高球和杀球。
学习后场固定点位吊球。

CONTENT-3：抽球

抽球，是将在身体左右两侧，肩膀以下，腰部以上的球平扫过去。有正手抽球和反手抽球有两种。

CLASS 4

CONTENT-3：跑动回球

学习结合步伐的跑动接发球。这一部分属于专项点位训练，需要一个一个动作、一个一个点位的喂球。
主要内容：

  • 6点跑动回球
  • 跑动后场吊直线

CONTENT-2：挡网

主要内容：

  • 平抽快挡
  • 泄力挡网

CONTENT-3：杀球

杀球，是指将对手击出的球在尽可能高的击球点斜压下去。这种球力量大，弧线直，落地快，对对手的威胁很大。它是主要的进攻技术。一般分为正手杀球、反手杀球。更细的分为：点杀、劈杀、霸王杀、土豆杀、跳杀、辣子鸡的章鱼杀、蛋总的头顶断腕点杀。
主要内容：

  • 霸王杀
  • 土豆杀
  • 点杀

CLASS 5

学习网前的正反手搓、勾、推球技术。

CONTENT-1：搓球

推球，是将网球从对手推至对手后场的两个底角。球飞行的弧线更低更平，速度更快。
搓球，是用球拍摩擦球托的左侧或右侧下部和底部，使球旋转并向右或向左滚过球网。搓球有正手搓球和反手搓球。

主要内容：

  • 正/反手平搓放网
  • 反手收搓放网
  • 正手展搓放网

CONTENT-2：推球

主要内容：

  • 网前平推
    • 推斜线 - 抢蹬上前，能充分发力，球速快，适合进攻打法，对体能和速度有一定要求，但有可能会被快速平抽直线后场大空位。
    • 推直线 - 抢蹬上前，需要让出击球身位，球速相对斜线慢，且人和球在一条直线上，不容易形成被动，适合防守反击打法。
    • 假推真放 - 假动作，骗对手重心，逼对手起球，适合控制打法。
  • 弹推

CONTENT-3：勾球

主要内容：

  • 正手勾对角
  • 反手勾对角

CLASS 6

CONTENT-1：突击

突击和杀球是羽毛球运动中两种常见的技术动作，它们的主要区别在于球的飞行速度和攻击方式。

突击：突击通常是在对方回球的一瞬间，利用快速的步伐移动到合适的位置，然后用球拍将球击打到对方场地的后方。突击的特点是速度快，攻击的突然性较强，让对方来不及反应。

杀球：杀球则是一种更为直接和力量型的攻击方式，通常是在对方回球较高或较平的时候，利用强大的力量将球击打到对方场地的前方。杀球的特点是力量大，球的飞行速度快，攻击的力度强，给对方造成的威胁更大。

总结，突击强调的是速度和突然性，而杀球则更注重力量和攻击的力度。

主要内容：

  • 头顶突击
  • 中国跳突击
  • 马来步突击

CONTENT-2：基础组合球路

学习组合球路，就是基础步法 + 6点位 + 回球。
主要内容：

  • 拉吊上网
  • 杀上网

CLASS 7

学习反手技巧，学习反手握拍变化。了解反手球落点划分：身前、身侧、身后。

CONTENT-1：

身前球

  • 反手拧瓶盖抽球
  • 反手平档

CONTENT-2：

身侧球

  • 手握斜棱，侧身过渡网前

CONTENT-3：

身后球

  • 向后转身，手握侧棱，反手高远、反手过渡网前、反手过渡斜线

CONTENT-4：

反手杀

CLASS 8

CONTENT-1：进阶吊球

  • 劈吊（渣男大斜线）
  • 滑板

CONTENT-2：进阶杀球

  • 跳杀

CONTENT-3：接杀

  • 接杀挑后场
  • 接杀分球

CLASS 9

实战课
主要内容：

  • 实战接发观察对手习惯，活动发球战术
  • 单/双打实战陪练
