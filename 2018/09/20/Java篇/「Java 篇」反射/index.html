反射

动态获取指定类以及类中的内容(成员)，并运行其内容。

反射技术

应用程序已经运行，无法在其中进行new对象的建立，就无法使用对象。这时可以根据配置文件的全类名去找对应的字节码文件，并加载进内存，并创建该类对象实例，这就需要使用反射技术完成。
反射作用的位置如下图所示：
简单地说就是反射可以获取代码的字节码对象，通过该字节码对象可以获取和设置信息。

获取class对象的三种方式

获取字节码对象通常有几下三种方式：

  1. Class.forName（“全包名”）

  2. 每一个类型都具备一个class静态属性，通过该属性即可获取该类的字节码文件对象。

  3. 通过对象具备的getClass方法(源于Object类的方法)

反射创建实例对象

创建实例对象有两种，无参构造创建实例对象和有参构造创建实例对象。Class类的newInstance()方法使用的是该类的无参构造创建对象，如果一个类没有无参的构造函数，就不能这样创建对象了，可以调用Class类的getConstructor(String.class,int.class)方法获取一个指定的构造函数然后在调用Constructor类的newInstance(“Maple”,26)方法创建对象。

  1. 通过无参构造创建实例对象

  2. 通过有参构造创建实例对象

反射获取字段

Class.getFiled(String)方法可以获取类中的指定字段(可见的)，如果是私有的可以使用getDeclaredField(“name”)暴力反射方法强行获取，通过set(obj,”Maple”)可以设置对象上该字段的值；如果是私有的需要先调用setAccessible(true)设置访问权限，用获取的指定的字段调用get(obj)可以获取指定对象中该字段的值。

反射获取方法

Class.getMethod(String,Class…)和Class.getDeclaredMethod(String,Class…)方法可以获取类中的指定方法，invoke(Object,Object)可以调用该方法。
通过获取无参方法步骤:

  1. Class.getMethod(“eat”)
  2. invoke(obj)

通过获取有参方法步骤:

  1. Class.getMethod(“eat”,int.class)
  2. invoke(obj,10)
