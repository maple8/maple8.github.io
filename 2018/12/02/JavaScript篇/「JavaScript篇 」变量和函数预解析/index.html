JavaScript 变量和函数预解析

大多数编程语言的变量和方法的使用规则都是先定义后使用，否则就会出现异常。JavaScript具有预解析的功能，能够打破这种规则，可以在变量和函数定义之前就使用该变量和函数（此时的变量值为undefined）。
JavaScript 代码的解析过程分为两个阶段，先是编译阶段，然后执行阶段。
在编译阶段会将用function定义的函数提前，并且将用var定义的变量 声明 提前，并将它赋值为undefined。相当于在代码一开始就创建了一个未初始化的变量（只是效果类似创建了一个变量，实际上没有创建变量）。
JavaScript的预解析功能可以让开发者在定义方法之前调用方法，也可以在定义变量之前调用变量，但是变量的值为undefined。
