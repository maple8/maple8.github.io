使用 ping 诊断网络

Posted on 2018-10-20 | In 计算机操作系统 |

ping 是 Windows、Unix 和 Linux 系统下的一个命令。ping 也属于一个通信协议，是 TCP/IP 协议的一部分。利用“ping”命令可以检查网络是否连通，可以很好地帮助我们分析和判定网络故障。

实际应用中，检查某个计算机的网络是否正常，我们常说“ping一下这个计算机”，就是使用上述方法。执行步骤如下：

  1. 获取计算机的IP地址

  • Windows 使用 Win + R 快捷键打开运行，然后输入 cmd，打开终端，然后输入 ipconfig，便可看见计算机的连接在网络中的IP地址

    只有计算机连接到网络中才有IP地址，如果没有链接网络是没有IP地址的，IP地址就是唯一在网络中标识某个计算机网卡地址的存在。

  • Linux 使用 Ctrl + Alt + T 快捷键打开终端，然后输入 ipconfig，便可看见计算机的连接在网络中的IP地址

    注意：ipconfig 和 ipconfig
