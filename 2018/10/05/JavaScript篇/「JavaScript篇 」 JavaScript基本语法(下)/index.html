对象

对象（Object）是面向对象编程的核心概念，它是已经命名的数据集合，也是一种比较更复杂的数据结构。

数组

对象是无序的集合，而数组（Array）是一组有序数据集合，数组的大小不是固定的，可以动态的增加和删除元素。它们之间可以相互转换，但是数组拥有大量方法，适合完成一些复杂的运算。JavaScript中的数据和Python中的数据的概念类型，数组中可以存放任意类型的对象。而Java中的普通数组存放的是基本数据类型，且数组的长度不能改变。集合容器，如List，Set，Map等，使用时都会使用泛型指定存储元素的对象类型。

注意：JavaScript数组的索引不能为负整数；Python中可以，如：arr[-1] 表示数组arr的最后一个元素。JavaScript中没有这种规则，但是即使这样写也不会出现语法错误，只是结果和预期的不一样。

数组的属性和方法

length属性
length是数组最常用的属性，返回数组的长度，但JavaScript的length属性还有一种特殊的功能，可以通过改变数组的length属性来改变数组的大小。

数组的原始长度为5，通过重新设置数组的长度为5-1=4，原始的数组的长度也减小了1，并将最后一位的元素删除了。可以看出，改变数组的长度属性时，会自动在数组的末尾增加和删除元素，以实现改变数组的大小。
join() 方法
将数组成员通过一个分隔符合并成字符串。

split()方法
使用split()方法可以将字符串分割成一个字符串数组，该方法有两个参数：第一个参数是指定用于分割的分隔符；第二个参数（可选）指定分割之后需要返回的数组的长度。

push() 和 pop() 方法
从数组最后增加或删除一个或多个元素，此方法类似与栈结构的压栈和弹栈，不过区别是压栈和弹栈操作每次只能操作一个元素。

unshift()和 shift() 方法
从数组前面增加或删除一个或多个元素。

indexOf() 方法
返回数组中元素第一次出现的索引值.

splice() 方法
在数组中的指定位置插入或删除成员。
该方法的第一个参数是指定插入的起始位置，第二个参数指定要删除的元素的长度，第三个参数（可选）是要插入的元素；返回值是从数组中删除的元素组成的数组。

concat()方法
连接数组，该方法能够将方法中的参数追加到指定的数组中，形成一个新的连接数组。

如果concat()方法中的参数包含数组，那么就将该数组展开后添加到指定的数组中。

reverse() 方法
将数组反转，在将字符串中个字符反转的时候会使用该方法。该方法是在原始数组的基础上进行操作的，不会新建一个新的数组。

多维数组

多维数组指的是数组的成员也是数组的数组，通过连续的方括号访问数组中的元素，其他的方法和一维数组是相同的。

批量操作数组中的数据，需要用到循环语句。

函数应用

调用函数使用小括号运算符来实现： 函数名.();
函数的参数分为两种：实参和形参。
实参：就是当函数被调用时，传给函数的参数，被称为实参；
形参：就是在函数定义的时候，传递给函数的参数，被称为形参，即形式上的参数；
匿名函数传参：

在括号运算符内部可以包含多个参数列表，参数之间通过逗号进行分割。函数的返回值没有类型的限制，它可以返回任意类型的值。一个函数可以返回包含多个return语句，但是在调用函数的只有第一个return语句被执行，而且该return语句后面的表达式的值作为函数的返回值被返回，return语句后面的语句会被忽略掉。

一般情况下，函数的形参和实参个数是相等的，但是在JavaScript中没有规定两者必须相等。如果形参的个数大于实参的个数，则多出的形参值为undefined；相反的，如果实参个数大于形参，则多出的实参就无法被形式参数访问，从而被忽略掉。

和其他编程语言一样，JavaScript的函数也可以作为参数和值。函数实际上也是一种结构复杂的数据，因此可以将它作为值赋予给其他的变量。
函数作为值
以下实例中将函数当做一个值赋予给变量 a,然后利用小括号来调用这个函数。函数是引用数据类型，将函数赋值给变量 a,则a就是这个函数的一个引用，在内存中指向了这个函数的位置，然后通过这个引用和小括号调用函数，并传参，最后得到函数执行的返回值，这个概念和Python中的方法和引用是一样的。

函数作为参数
函数可以作为参数传递给另外一个函数，也可以作为返回值，通过这个方式可以增强函数的应用能力。将函数作为参数传递，严格意义上来讲是将函数的引用作为参数进行传递。
