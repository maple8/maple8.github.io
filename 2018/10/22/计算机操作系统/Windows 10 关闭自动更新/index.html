Windows 10 关闭自动更新

Posted on 2018-10-22 | In 计算机操作系统 |

Windows 10 操作系统是默认自动更新的，一旦有新的功能更新和安全更新都会后台自动下载，然后会有弹框提示何时更新，更在使用快捷键（Alt + F4）关机的时候多出几个选项：关机并更新；重启并更新等等。
很多用户不喜欢Windows 10 操作系统自动更新功能，因为这个更新操作不会考虑计算机的硬件性能，所以会导致某些不支持升级更新的硬件在操作系统自动更新之后会出现运行中的卡顿现象，严重影响了用户的体验。而且自动更新动能会下载大量的更新软件包，几乎和源操作系统差不多大，多达20~30G。这对于那些系统盘没有充足的容量的用户来说是很麻烦的事情，毕竟系统盘不像其他的动态磁盘一样能够通过吸盘管理器随时扩张容量。以下提供两种禁止win 10 自动更新的方法：

方法一：
把网络设置为“按流量计费的连接”这是微软为流量套餐有限制的用户提供的，既然要更好的控制数据使用了，就随心所欲自动升级。

  1. 在 Win10 任务栏右侧的网络图标，点击鼠标右键；
  2. 然后选择“打开网络和Internet设置”；
  3. 接下来选择“更改连接属性”；
  4. 最后选择“开启流量计费”就可以了。

方法二：

  1. 打开 “运行”菜单：
    win + 运行 或 快捷键 Windows+R
  2. 打开“服务”：
    输入“services.msc”并回车 或 在计算机上鼠标右键 -> 管理 -> 服务和应用程序 -> 服务
  3. 设置 Windows Update
    在右边区域中找到 Windows Update，双击进去，常规选项卡中设置：启动类型 -> 禁止；服务状态 -> 停止；恢复选项卡设置：将第一次失败，第二次失败，后续失败都设置为无操作；然后点击应用，确定。
    在服务中双击Windwos Update服务，在常规下将启动类型改为“禁用”，在恢复下将失败后改为“无操作”。

  4. 打开控制面板 -> 程序 -> 程序和功能，找到并选中易升软件，右键点击卸载(这个不一定都有)。
    在C:\Windows的位置，删掉C:\Windows目录下面UpdateAssistant文件夹。
