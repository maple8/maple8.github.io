JavaScript简介

JavaScript是一种轻量级、解释型的Web脚本语言，是运行在浏览器端的脚步语言，JavaScript主要解决的是前端与用户交互的问题，包括使用交互与数据交互。 JavaScript是浏览器解释执行的，由于所有的浏览器都已经嵌入了JavaScript引擎，所以不需要安装任何的运行环境便可对其进行开发。

前端三大语言

  1. HTML：页面结构，主要针对网页的内容
  2. CSS：页面表现，元素大小、颜色、位置、隐藏或显示、部分动画效果
  3. JavaScript：页面行为，部分动画效果、页面与用户的交互、页面功能

JavaScript嵌入页面的方式

  1. 行间事件（主要用于事件）,一般不推荐这样的用法，书写很容易出现错误。

  2. 页面script标签嵌入，可以直接将JavaScript代码直接放入网页文档中。

  3. 外部引入，通过属性src指定文件的URL（统一资源定位符）即可，使用外部JavaScript文件，能够增强JavaScript模块化开发的程度，提高代码的复用性。

一般来说，JavaScript代码可以嵌入到网页中的任何位置，但考虑到HTML文档的DOM结构模型规范性，建议将JavaScript脚本写在<head>...</head>标签和<body>...</body>标签之间。每个网页都可以包含多个JavaScript的<script>标签，每个标签包含的代码被称为JavaScript的脚本块，一般将相同或相近的代码放在一个脚本块中，而不同功能的代码分别放在不同的脚本块中会更加便于管理。适当时可以将通用的脚本模块放在单独的JavaScript脚本块中。

<script>标签包含了两个属性：
type：设置标签包含脚本属于什么文本类型，如text/javascript属性值表示JavaScript代码类型的文本，默认的值就是text/javascript，所以可以省略。
language：设置标签包含脚本属于什么语言类型，JavaScript属性值表示JavaScript语言类型，该属性值主要适应早起浏览器的解析，因为他们会忽略掉type属性声明。

变量和数值类型

JavaScript 是一种弱类型语言，在定义变量的时候不需要声明变量的数值类型，一个变量可以存储任何类型的值，不过这不等于JavaScript不区分数据类型，JavaScript的变量类型由它的值来决定。 定义变量需要用关键字 ‘var’。

数值类型

在JavaScript中，数据存在两种截然不同的存储方式，一种是直接存储方式，称为值类型数据；另外一种是存储数据的在内存空间中的地址来间接保存数据，称为引用型数据。
JavaScript中有3种基本数据类型、2中特殊数据类型和3种复杂数据类型。
基本数据类型：

  1. number： 数字类型
  2. string： 字符串类型
  3. boolean： 布尔类型 ，值为true 或 false

特殊数据类型：

  1. undefined： undefined类型，变量声明时未初始化，它的值就是undefined
  2. null： null类型，表示空对象，如果定义的变量将来准备保存对象，可以将变量初始化为null,在页面上获取不到对象，返回的值就是null

复杂数据类型：

  1. Object：对象类型
  2. Array：数组类型
  3. Function：函数类型

命名

命名规范：

  • JavaScript严格区分大小写；
  • 名称中的字符必须是字母、数字、下划线（_）或者美元符号（$），且第一个字符不能是数字；
  • 名称不能是JavaScript的关键字或保留字；

匈牙利命名风格：
对象o： Object 比如：oDiv
数组a： Array 比如：aItems
字符串s： String 比如：sUserName
整数i： Integer 比如：iItemCount
布尔值b： Boolean 比如：bIsComplete
浮点数f： Float 比如：fPrice
函数fn： Function 比如：fnHandler
正则表达式re： RegExp 比如：reEmailCheck

类型转换

JavaScript数值是不区分整型和浮点型，所有数值都是以浮点型数值来表示的。
数值类型–>字符串类型
使用toString()方法可以将数值类型转换为字符串类型。

将数值转换为字符串有一种便捷方法：使用数值和空字符串相加，即可把数值转换为字符串。

数值类型–>布尔类型
布尔类型数据只有两个值：true和false，分别表示逻辑的真和假。布尔值多用在比较运算中，或者作为条件语句或运算符的条件使用。
要把数值类型转换为布尔类型，在值得前面添加两个感叹号即可：

任何非0数字转换为布尔值后为true，而0转换为布尔值为false。
任何非空字符串转换为布尔值后为true，而空字符串转换为布尔类型的false。

布尔类型–>数值类型
将布尔类型转换为数值类型，true为1，false为0。便捷方法是将布尔类型值乘以数值1即可。

布尔类型–>字符串类型
布尔类型转换为字符串类型，则true–>”true”, false–>”false”。便捷方法是和空字符串相加。

隐式转换 “==” 和 “-”
自动类型转换，JavaScript中的 == 运算符会将两边的数值进行自动类型转换，在自动类型转换的过程总，JavaScript一般遵循：如果某个类型的值被用于需要其他类型的值的环境中，JavaScript就自动将这个值转换成所需要的类型。

浮点数溢出问题
JavaScript在执行数值运算的时候，常会出现浮点数溢出问题，如：0.1+0.2不等于0.3
alert(parseFloat(‘5.6’)+2.3); // 弹出7.8999999999999995
alert(0.1+0.2); //弹出 0.3000000000000004
解决方法：先将结果扩大100倍，最后再取整。
alert((0.1100+0.2100)/100); //弹出0.3
alert((parseFloat(‘5.6’)100+2.3100)/100); //弹出7.9

JavaScript表达式语句

javascript在表达式的尾部添加一个分号就会形成一个表达式语句，JavaScript默认的独立的一行表达式也是表达式语句，表达式语句是最简单、最基本的语句。这种语句一般按着从上到下的顺序依次执行。

选择语句和运算符

通过条件来控制程序的走向，就需要用到条件语句。JavaScript的条件语句有：if，if…else, if …else if … else if… else…, switch，针对不同的需要进行选择。
运算符

  • 算术运算符： +(加)、 -(减)、 *(乘)、 /(除)、 %(求余)
  • 赋值运算符：=、 +=、 -=、 *=、 /=、 %=
  • 条件运算符：>、>=、<、<=、==、!=、&&(而且)、||(或者)、!(否)
  • 三元运算符：？：，相当于if…else…

    还有一个特殊的运算符===,这个运算符和两个等号运算符有点不一样，JavaScript在使用两个等号运算符进行运算的时候会自动对运算符左右两边的数值进行类型转换，而三个等号运算符是不会自动进行类型转换的，该运算符一开始就判断左右两边的数值是否是相同类型的，如果不是相同类型的就直接返回false，不用再继续执行判断内容了。

字符串

JavaScript字符串有Unicode字符、数字、标点符号等组成的字符序列，字符串必须使用单引号或是双引号包括起来，和Python中一样，单引号中可以包括双引号，双引号中可以包括单引号，所有的字符应该在同一行内书写。
字符串处理方法
1、字符串合并操作：“ + ”

2、parseInt() 将数字字符串转化为整数

3、parseFloat() 将数字字符串转化为小数

使用parseInt()和parseFloat()需要注意的地方，需要转换的字符串应该是数字字符串，如果字符串中包含了非数字，返回的结果如下：
alert( parseInt(‘123abc’) ); // 弹出123
alert( parseInt(‘abc123’) ); // 弹出NaN

将字符串转换为数值还有一种常见的便捷方法，让字符串和数值1相乘，即可把字符串转换为数值。

4、split(sStr) 使用给定的字符串把一个字符串分隔成字符串组成的数组。特别的是给定一个空字符串””，可以将字符串的所有字符转换成一个字符数组。

5、charAt(index) 根据索引值获取字符串中的某一个字符。字符串操作和数组操作类似，都可以通过索引来获取其中的元素。

6、indexOf() 查找字符串是否含有某字符串，并返回该字符串第一次出现的位置索引。

7、substring() 截取字符串 用法： substring(start,end)（不包括end）

8、toUpperCase() 字符串转大写

9、toLowerCase() 字符串转小写
