获取元素

要在HTML页面中获取某个标签中的属性值，用于后续的操作，获取元素的常用方法有两种：

方法一：document.getElementById();

使用内置对象document上的getElementById方法来获取页面上设置了 id 属性的元素，获取到的是一个 html对象，然后将它赋值给一个object类型的变量，比如：

第一种方法：将javascript放到页面最下边
不过这种方法不推荐使用，格式上不太严谨。

这样一来，JavaScript获取元素的语句就会在整个页面加载完后才执行，就不会出错了。

方法二：document.getElementByTagName();

使用内置对象document上的getElementsByTagName方法来获取页面上的某一种标签，获取的是一个选择集，不是数组，但可以使用数组下标的方式操作选择集里面的标签元素。

通常一个HTML页面中同一个标签名会出现多次，所以该方法获得的标签对象是一个选择集，而不是单个标签对象
