openpyxl 模块

名词解释：

  1. workbook：工作簿，整个Excel文件；
  2. worksheet：表单，Excel文件中的单元文件，默认第一张表单是活跃表单；
  3. row，表单的行数，从1开始，1，2，3，…；
  4. columu，表单的列数，从A开始，A，B，C，…；
  5. 单元格：表单中的某一格，通过行号和列号来确定，如：A1…；
  6. 三个重要对象：工作簿对象wb，表单对象ws，单元格对象cell；

工作簿对象(wb)的常见属性和方法：

属性名 含义
wb.sheetnames 属性 以列表形式返回工作簿中的所有表单名
wb.active 属性 获取活跃表单，默认是第一个表单，或者使用get_sheet_by_name方法获取表单对象
wb[‘表单名’] 属性 通过表单名获取表单对象
wb.get_sheet_by_name(‘表单名’) 方法 同上，通过表单名获取表单对象
wb.create_sheet(title=None, index=None) 方法 在工作簿中创建一个新的表单，title参数表示表单名，index表示表单的索引
wb.remove_sheet(worksheet) 方法 从工作簿对象中删除表单，worksheet表示待删除的表单对象
wb.save(filename) 方法 保存工作簿对象到指定的Excel文件中，filename表示Excel文件名

  1. wb.sheetnames 属性

  2. wb.active 属性

  3. wb[‘表单名’]和wb.get_sheet_by_name(‘表单名’)

  4. wb.create_sheet(title=None, index=None) 方法

  5. wb.remove_sheet(worksheet) 方法

  6. wb.save(filename) 方法

  1. 工作簿对象是一个可迭代对象

表单对象(ws)的常见属性和方法：

属性名 含义
ws.title 属性 获取表单的名称
ws.max_row 属性 获取表单中使用的行数，也就是光标所到的最后一行，即使整行全是空的，也会被算入行数
ws.max_column 属性 获取表单中使用的列数，也就是光标所到的最后一列，即使整列全是空的，也会被算入列数
ws.rows 属性 获取所有行单元格的生成器对象
ws.columns 属性 获取所有列单元格的生成器对象
ws[‘单元格的坐标’] 属性 获取单元格对象，如：ws[‘A1’]，获取第一行第一列的单元格对象
ws.cell(row, column) 方法 通过下标获取单元格对象，行和列的下标都是从1开始
ws[‘行坐标’] 属性 以元组形式获取整行单元格对象，如：ws[‘2’]，获取第二行所有单元格对象
ws[‘列坐标’] 属性 以元组形式获取整列单元格对象，如：ws[‘A’]，获取第一列所有单元格对象
ws[‘行坐标:行坐标’] 属性 以元组形式获取切片单元格对象，如：ws[‘2:4’]，获取第二行到第四行的所有单元格对象
ws[‘列坐标:列坐标’] 属性 以元组形式获取切片单元格对象，如：ws[‘A:C’]，获取第一列到第三列的所有单元格对象
ws[‘行坐标:列坐标’] 属性
ws[‘列坐标:行坐标’] 属性		 通过切片获取贯穿行、列的多个单元格，如：ws[‘A1:C4’]，
获取从第一行第一列的单元格到第四行第三列的单元格之间的所有单元格对象
ws.iter_rows(min_row, min_col, max_row, max_col) 方法 表单生成器，获取指定行数和列数之间的所有单元格对象
ws.append(row) 方法 以列表的形式对单元格进行整行写入，row参数是一个列表对象

  1. ws.title属性

  2. ws.max_row属性 和 ws.max_column属性

  3. ws.rows属性 和 ws.columns属性

  4. ws[‘单元格的坐标’] 属性 和 ws.cell(row, column) 方法

  5. ws[‘行坐标’] 属性 和 ws[‘列坐标’] 属性

  6. ws[‘列坐标:列坐标’] 属性 和 ws[‘行坐标:行坐标’] 属性

  7. ws[‘行坐标:列坐标’] 属性 和 ws[‘列坐标:行坐标’] 属性

  8. ws.iter_rows(min_row, min_col, max_row, max_col)方法，表单的生成器方法，通过传入方法中的四个参数来确定单元格的范围，返回的是一个生成器对象

  9. 单元格的写入

单元格对象(cell)的常见属性和方法：

属性名 含义
cell.value 属性 获取单元格中的值，根据单元格值得类型返回对应的python类型的数据；如果是整数返回值类型int，如果是字符串返回类型为str，空的单元格值为None。
cell.row 属性 获取单元格所在的行数，从1开始
cell.column 属性 获取单元格所在的列数，从1开始
cell.coordinate 属性 获取单元格的整体坐标，如：’A1’

  1. cell.value 属性、cell.row 属性和cell.column 属性

  2. cell.coordinate 属性，获取单元格的完整坐标
