Python操作Excel之openpyxl

​ Python通过openpyxl可以非常方便的实现对Excel文件的操作，包括读取、写入、更改、计算、样式、表格等等都可以实现。但是Python作为一门脚本语言，其特点就是简单，实现方便，并具有大量的第三方开源包。对于Excel文件的操作没有什么特定的规律，针对不同的问题需要编写不同的功能脚本，而且同一个脚本再次使用的概率很小，所以相对来说也不好实现对常用的脚本进行封装的操作，所以，此处简单地实现几个功能代码段，虽然直接用于实际工作的概率很小，但其包括了openpyxl模块的常见基本属性和方法的操作，为后续改写脚本起到抛砖引玉的作用。
