Git 概述

Git 是一个开源的分布式版本控制系统，可以有效、高速的处理从很小到非常大的项目版本管理。 Git 是 Linus Torvalds 为了帮助管理 Linux 内核开发而开发的一个开放源码的版本控制软件。

    https://gitee.com/
  2. 国外git服务器地址–Github：
    https://github.com/

另外一种上传文件到远程仓库

上述将项目文件上传至远程仓库中的方法，必须要先创建一个同名的空项目，然后在码云端手动将项目文件拖拽到上传文件框中，但这一步骤在码云端是有限制的，只能拖拽最多20个文件，如果文件太多就不能使用该方法了。以下是一种无须创建本地空项目并拖拽到码云端的方法：
